# Equipment Database

Reference dimensions and visual signatures for military equipment identification.

## Sources (Open Data)
- **CSIS Missile Threat**: Missile dimensions/specs (https://missilethreat.csis.org/)
- **Jane's / IHS Markit**: Defence equipment reference
- **Wikipedia**: Equipment specifications (cross-referenced)
- **Naval Vessel Register**: US Navy ship dimensions

## Example Entry
```json
{
  "id": "type-094",
  "name": "Type 094 (Jin-class SSBN)",
  "category": "submarine",
  "country": "CN",
  "length_m": 135,
  "beam_m": 12.5,
  "visual_signatures": {
    "sail_shape": "rounded",
    "missile_compartment": "visible_hump",
    "color_hints": "dark_hull"
  }
}
```
